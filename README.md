# Indicator Control using Arduino Nano

This project simulates a vehicle indicator control unit.

Features
- Left indicator
- Right indicator
- Hazard lights
- Button debounce
- Button hold detection
- 100 ms scheduler
- UART logging on serial monitor

Hardware
* Arduino Nano
* Push Buttons
* LEDs
* Resistors
* Breadboard

Software
* Arduino IDE
